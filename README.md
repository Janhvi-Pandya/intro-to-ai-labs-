# Intro to Artificial Intelligence ‚Äî Applied Labs

This repository contains a curated set of **applied AI labs** completed as part of an *Intro to Artificial Intelligence* course.  
The labs focus on **problem-solving, algorithmic reasoning, probabilistic modeling, and machine learning**, with an emphasis on **analysis, evaluation, and practical implementation in Python**.

Each notebook demonstrates a different AI paradigm, progressing from classical search to learning-based methods.

## Lab Overview

### **Lab 01 ‚Äî Search Algorithms & Constraint Satisfaction**
**File:** `ai-lab01-search-algorithms-csp.ipynb`

Implemented and analyzed classical AI search algorithms to solve structured problem spaces.  
Compared uninformed and heuristic-based approaches using performance metrics such as node expansion counts and frontier size.  
Applied constraint satisfaction techniques to systematically reduce search complexity.

**Key concepts:**
- Graph search algorithms  
- Heuristic evaluation  
- Constraint satisfaction problems (CSPs)  
- Algorithm performance analysis  

---

### **Lab 02 ‚Äî Heuristic Search & Performance Optimization**
**File:** `ai-lab02-heuristic-search-optimization.ipynb`

Designed and optimized heuristic-driven search strategies to improve computational efficiency.  
Evaluated trade-offs between optimality and performance by comparing heuristic quality, search depth, and resource usage across multiple strategies.

**Key concepts:**
- Heuristic design  
- Search optimization  
- Complexity and efficiency analysis  
- Algorithmic trade-offs  

---

### **Lab 03 ‚Äî Probabilistic Reasoning & Bayesian Modeling**
**File:** `ai-lab03-probabilistic-reasoning-bayesian-models.ipynb`

Applied probabilistic reasoning techniques to model uncertainty and support decision-making under incomplete information.  
Implemented Bayesian inference methods and analyzed probabilistic outcomes to assess model behavior and decision accuracy.

**Key concepts:**
- Probability theory  
- Bayesian inference  
- Uncertainty modeling  
- Statistical reasoning  

---

### **Lab 04 ‚Äî Machine Learning Classification & Evaluation**
**File:** `ai-lab04-machine-learning-classification.ipynb`

Built supervised machine learning classification models to learn patterns from labeled datasets.  
Performed data preprocessing, feature analysis, and model evaluation to compare performance and generalization across approaches.

**Key concepts:**
- Supervised learning  
- Classification algorithms  
- Feature analysis  
- Model evaluation and validation
